/news/category/create - form for filling out a category.
                        Will use data from ini file do determine which kind of data can be used.
/news/category/create?insert - the form will be checked for errors, then inserted. If any errors,
                               tell the user what and why.
/news/category/id/#   - edit the category with the id.
/news/category/id/#?delete   - delete this category.
/news/category/path   - edit the category with the canonical path.
/news/category        - give info about orphaned categories, etc.
